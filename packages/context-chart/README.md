# @baggiiiie/pi-context-chart

A pi package that adds the `/context-chart` command.

## Install

```bash
pi install /path/to/pi-stuff/packages/context-chart
```

Or after publishing:

```bash
pi install npm:@baggiiiie/pi-context-chart
```

## Usage

```text
/context-chart
/context-chart close
```

## Notes

- Requires pi
- Requires Glimpse to be installed where Node can resolve it, or `GLIMPSE_PATH` set to `.../glimpseui/src/glimpse.mjs`
